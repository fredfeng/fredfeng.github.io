# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is Yu Feng's academic homepage hosted via GitHub Pages at `fredfeng.github.io`. It is a static site originally generated by Hugo (v0.41) with the `hugo-academic` theme, but is now maintained as pre-built static HTML — there is no Hugo build step.

## How to Make Changes

All content edits are made directly in the static HTML/CSS files. There is no build system or static site generator to run. Push to `master` to deploy via GitHub Pages.

- **Main page**: `index.html` — contains all homepage sections (about, students, publications, awards, service, contact) as a single-page layout with anchor navigation
- **Styles**: `styles.css` — site-wide custom CSS
- **JS**: `js/hugo-academic.js` — theme JavaScript

## Site Structure

- `index.html` — the main homepage (single HTML file with all sections)
- `publication/` — individual publication pages (one folder per paper, each with `index.html`)
- `img/` — profile photos and icons
- `Trinity/` — documentation site for the Trinity/Tyrell program synthesis framework (Sphinx-generated, do not edit)
- `sypet/` — project page for the SyPet tool (standalone static site)
- `pldi17/` — Keynote presentation export for PLDI 2017 talk

## Key Patterns

- The site uses Bootstrap 3.3.7 for layout and Font Awesome 4.7 for icons
- CDN-hosted dependencies (highlight.js, academicons, jQuery, Bootstrap JS)
- Publications section in `index.html` lists recent papers; `publication/` has full details
- The `.nojekyll` file disables Jekyll processing on GitHub Pages
