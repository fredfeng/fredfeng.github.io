assertArg(node: tyrell.dsl.node.Node, args: List[Any], index: int, cond: Callable[[Any], bool], capture_indices: Iterable[int] = []) → None[source]¶

Check the value of index-th argument against cond. If cond is not met, raise AssertionViolation. If the assertion refers to the value(s) of other arugments, the indices of those arguments should be passed to capture_indices as a list of integer. By default, this list is empty.

eval(prog: tyrell.dsl.node.Node, inputs: List[Any]) → Any[source]¶

Evaluate a DSL prog on input inputs. The output is returned. This is a covenient wrapper over eval_step that repeatedly invoke the generator until we get the final result.
